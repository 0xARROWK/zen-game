Bienvenue dans le jeu Zen L'initié !

Veuillez lire attentivement le manuel utilisateur afin d'assimiler toutes les fonctionnalités disponibles.



IMPORTANT :

Si vous n'arrivez pas à lancer le jeu, utilisez le fichier build.xml
Dans un terminal, pLacez-vous dans le répertoire ou se trouve le build.xml et tapez la commande "ant"
Le jar sera ainsi créé avec votre version de java.



Ce fichier va vous aider à lancer l'application correctement.

Vérifiez que le dossier saved se trouve dans le même répertoire que le fichier Zen.jar
IL s'agit de la seule condition nécessaire au bon fonctionnement de l'application.

Si vous souhaitez déplacer le jeu, il suffit de déplacer le répertoire dans lequel se trouve le fichier
Zen.jar et le dossier saved.

Ouvrez un terminal et placez-vous dans le dossier où se trouve le fichier Zen.jar

Afin de pouvoir démarrer le jeu, deux commandes sont disponibles.

La première, avec l'argument "c", permet de lancer le jeu en mode console :

java -jar ./Zen.jar c

La seconde, avec l'argument "g", permet de lancer le jeu en mode console :

java -jar ./Zen.jar g

Si vous êtes sous windows, vous pouvez double-cliquer sur le fichier Zen.jar

Le dossier saved contient les sauvegardes des parties précédentes
